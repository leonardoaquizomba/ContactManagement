# ContactManagement

Para execução da aplicação devera seguir os seguintes passos.
1 - Baixar o projecto.
2 - Restaurar o pacotes atraves do comando: dotnet restore
3 - Editar o arquivo appsettings.json informando a string de conexão em MariaDB: <stringconexão>
4 - Caso não existe a pasta Migrations executar o comando: Add-Migration init e posteriomente Update-Database
5 - Fazer o Debugging do projecto
6 - Podera criar uma conta atravez do menu Register e fazer o Login.
